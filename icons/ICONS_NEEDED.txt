ICONS NEEDED FOR CHROME EXTENSION
===================================

Create these 3 PNG files:

1. icon16.png (16x16 pixels)
2. icon48.png (48x48 pixels)  
3. icon128.png (128x128 pixels)

DESIGN:
- Use IronCV logo (red target symbol)
- Transparent background
- Red color: #C41E3A
- Simple, recognizable at small sizes

HOW TO CREATE:

Option 1: Use IronCV Logo
- Export logo from Figma/design tool
- Resize to 128x128, 48x48, 16x16
- Save as PNG with transparency

Option 2: Quick Placeholder
- Go to https://www.favicon-generator.org/
- Upload IronCV logo
- Download generated icons
- Rename and place in this directory

Option 3: Use Emoji
- Temporary solution for testing:
- 🎯 emoji works well as placeholder
- Convert emoji to PNG online
- Resize to needed dimensions

For now, the extension will work without icons (Chrome shows default).
Add icons before publishing to Chrome Web Store!
